# CosmoExplorer



##  About 

this website allows you to watch real time photos from the NASA database and see data from Mars.
this project is for every person who is passionate about astronomy.

## How to install 
- download the source code
- ensure thst nodeJS and NPM are installed in your machine
-open your terminal and install dependecies running
```
npm i
```

you can start exploring the website running
```
npm run dev
```
